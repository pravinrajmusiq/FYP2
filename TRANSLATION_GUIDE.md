## ЁЯФД How Translation System Works

### 1. **Automatic Detection**
```javascript
const currentLang = localStorage.getItem('kioskLanguage') || 'en';
```

### 2. **Title Rendering Logic**
```javascript
if (currentLang === 'en' || currentLang === 'ms') {
    // Show IMAGE font
    titleElement.innerHTML = '<img src="...font.png">';
} else {
    // Show TEXT translation (Tamil & Chinese)
    titleElement.textContent = t('translationKey');
}
```

### 3. **Text Element Translation**
All elements with `data-translate` attribute are automatically translated:
```html
<button data-translate="back">BACK</button>
```

### 4. **Dynamic Content**
JavaScript dynamically updates translations:
```javascript
element.textContent = t('translationKey');
```
